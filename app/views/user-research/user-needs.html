The original GOV.UK accounts discovery that took place in 2019 identified user needs and set out the original justification for the work.

Whilst our project was not aimed at gathering user needs as part of discovery style research, we have found it useful to document user needs along the way.

These user needs have enabled valuable discussions across the project. By putting user needs first, we can discuss which features best meet these needs, and also identify whether any of the provisional ideas for features could adversely impact users.

This has all supported conversations around risk, impact and future service design.
