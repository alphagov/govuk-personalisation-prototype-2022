Feature was discussed but not included in the card sort

Whilst there were a number of use cases where this feature could add value to services, the level of concern and challenges raised around this feature outweigh the benefit that could be delivered.

  • Concerns around transparency with the end user
  • Validity of the information sources (government is held to high data standards - this won't be the same for external data sources) Government is held to a higher standard, there's no parity of safeguarding.
  • A huge amount of education would need to take place with the end user
  • The long tail effect - users share information because it is convenient but they don't think about the long tail effects
