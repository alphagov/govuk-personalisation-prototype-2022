Some users believe that their data is shared across all government departments

This finding is not unique to this piece of work. It supports previous research that has taken place within the GOV.UK accounts programme.

We believe that the current branding and wording of the GOV.UK account confuses users, as they don’t understand the difference between the website and the account. They think they are the same thing which is reflected in their language.

This may result in end user misunderstandings of the role of the GOV.UK account within cross-gov transactions.

In later rounds of research, we tested a global push ‘warning’ which explained to users that their data would not automatically be shared across all departments. Whilst this was generally understood, in many cases we had to draw the users’ attention to the content as they did not notice it when scanning the page. We believe that this user expectation cannot be managed by content alone.

WarningFuture design should take user expectations into account.

Examples from user research:

“I’m assuming it would be shared across anything else that comes under GOV.UK. If I use other services across GOV.UK it would be shared across the board.” from phase 1, round 1

“When I sign up I put my details in with my address details. I would assume that because it already has my address it would link up with my existing claim. Because there are a few things under GOV.UK you can do on their website like tax for the year, housing benefit, because it is all under GOV.UK I’d like to think it would link rather than having to put all the details in again.” from phase 2, round 1.

“I believe it [data] should automatically update. If you’ve made the change electronically there should be nothing stopping it from automatically updating.” from phase 2, round 2

“I would hope my address would be updated across all services if I changed it.” from phase 2, round 4

“It’s all stored within the GOV.UK website.” from phase 2 round 3

“You use one website to access everything.” from phase 1, round 2

“It [data] will be used across all the different websites to speed up the process.” from phase 2, round 5
