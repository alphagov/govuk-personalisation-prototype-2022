A deep dive into core features that enable an MVP for X-Gov Services

We will be using our learnings from phase 1 and 2 to prioritise core features that enable an MVP that will explore:

  • The ability to save data in a GOV.UK account
  • A dashboard area that allows users to manage their data at a basic level
  • Basic functionality of automatic form filling using saved data within the GOV.UK account

These core features will also provide the basic functionality and capabilities in order to implement the remaining features within the roadmap for X-Gov Services.
